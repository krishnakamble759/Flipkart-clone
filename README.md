# Flipkart Clone

A fully responsive Flipkart clone built using modern web technologies. This project replicates the core features and design of the Flipkart e-commerce platform, including product listings, detail pages, and a functional shopping cart.

- **Responsive Design**: Optimized for mobile, tablet, and desktop viewing.

- **Search System**: Search for products across different categories.
- **Category Browsing**: Explore products through a structured category navigation.

## üì¶ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18.0.0 or higher)
- [npm](https://www.npmjs.com/)
- [Git](https://git-scm.com/downloads) (Required for pushing to GitHub)

### Troubleshooting `git : The term 'git' is not recognized`

If you see an error saying `git` is not recognized:
1. **Install Git**: Download and install it from [git-scm.com](https://git-scm.com/downloads).
2. **Restart Terminal**: After installing, close and reopen your terminal or VS Code.
3. **Environment Variables**: During installation, ensure the option "Git from the command line and also from 3rd-party software" is selected.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/krishnakamble759/flipkart-website.git
   ```

2. Navigate to the project directory:
   ```bash
   cd flipkart-website
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Build for production:
   ```bash
   npm run build
   ```

## üåê Deployment

This project is configured to be easily deployed to services like GitHub Pages, Vercel, or Netlify.

### GitHub Pages (via GitHub Actions)

To deploy to GitHub Pages, you can use the static build in the `dist` folder. It is recommended to use a GitHub Action for automatic deployment whenever you push to the `main` branch.

1. Go to your GitHub repository **Settings** > **Pages**.
2. Under **Build and deployment** > **Source**, select **GitHub Actions**.
3. Use a "Static HTML" or "Vite" template to automate the build and deploy process.
